# Инкрементальная маршрутизация в динамически реконфигурируемых графах

[![Python 3.9+](https://img.shields.io/badge/python-3.9+-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Код для статьи **«Инкрементальная маршрутизация в динамически реконфигурируемых графах сетей связи и промышленного интернета вещей»**.

## Описание

Реализация инкрементального алгоритма обновления кратчайших путей с механизмом стабилизации (порог η). Код воспроизводит все эксперименты, описанные в статье:

1. Сравнение вычислительной сложности с полным пересчётом Дейкстры и A*
2. Анализ влияния порога η на дрожание маршрутов (Churn)
3. Анализ влияния порога η на качество путей (Stretch)
4. Компромисс «Качество vs Стабильность»
5. Сравнение размера области влияния на Mesh и AMI топологиях
6. Границы применимости при серии последовательных отказов

## Установка

1. Клонируйте репозиторий:
   ```bash

   cd incremental-routing-simulator
2. Создайте виртуальное окружение (опционально):
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/macOS
   venv\Scripts\activate     # Windows
4. Установите зависимости:
   ```bash
   pip install -r requirements.txt

## Запуск
   ```bash
   python code_of_article.py
   ```
Скрипт выполнит все 4 эксперимента и сохранит графики в папку ./graphs/.

Ожидаемое время выполнения: от минуты (зависит от производительности компьютера).
